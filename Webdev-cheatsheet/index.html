The Command-Line

Navigation Commands

Command Meaning discription
$ ls List Lists all files and directories in the current directory
$ pwd Print Working Directory Outputs the name of the directory you are currently in
$ cd Change Directory Switches you into the directory you specify

Directory Commands

Command Meaning discription
$ mkdir Make Directory Creates a new directory within the selected directory or current working directory
$ touch touch Creates a file within the selected directory
$ rmdir Remove Directory Deleates a directory
$ rm Remove Can remove files - add -r to deleate a directory and everything in it
